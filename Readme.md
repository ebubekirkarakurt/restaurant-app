## Installation

Install my-project with npm

```bash
  git clone https://github.com/ebubekirkarakurt/restaurant-app.git
  cd restaurant-app
```
Install NPM packages
 
```bash
  npm install
```
Start metro server
 
```bash
  npx expo start

```
